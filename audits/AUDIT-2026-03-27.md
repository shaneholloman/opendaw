## 4. Architecture

### StudioService God Class (575 lines)

`packages/app/studio/src/service/StudioService.ts` conflates 15+ responsibilities:

1. Layout/workspace management
2. Timeline management (range, snapping, clips)
3. Menu population
4. Panel layout management
5. Spotlight data supply
6. Sample playback
7. Project profile service integration
8. Engine facade management
9. Audio device management
10. Cloud authentication
11. Code editor state
12. Room awareness coordination
13. Chat service integration
14. Recovery management
15. Keyboard lifecycle
16. Signal broadcasting
17. Console commands

Could be split into: LayoutService, TimelineService, AudioService, ProjectService, CollaborationService, EditorService.

### studio-core Mixed Responsibilities (14,437 lines, 156 files)

`packages/studio/core/` acts as a kitchen sink combining:

- Project management (`project/`)
- Engine/audio processing (`Engine*`, `Mixer.ts`)
- Import/export (`dawproject/`)
- Sample & soundfont management (`samples/`, `soundfont/`)
- MIDI/capture (`midi/`, `capture/`)
- Cloud integration (`cloud/`)
- UI rendering (`ui/`)
- Collaborative sync (`ysync/`)
- File handling (`ffmpeg/`)

Could be split into: studio-project, studio-engine, studio-io, studio-samples, studio-sync.
